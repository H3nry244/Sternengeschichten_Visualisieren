Kosmische Blockade!

Der Browser blockiert das Laden der episodes.json aus Sicherheitsgründen (CORS-Richtlinie), weil die Seite direkt als Datei geöffnet wurde.

Lösung: Starte einen lokalen Webserver! In VS Code einfach unten rechts auf "Go Live" (Live Server Extension) klicken oder im Terminal im Projektordner folgenden Befehl eingeben:

python -m http.server
